### 安装教程

1. 克隆项目到本地， `composer install` 
2. 配置数据库： `config/database.php` 
3. 导入数据库
4. 启动内置服务器： `php think run` ，然后你可以直接在浏览器里面访问：
```
http://127.0.0.1:8000/
```

后台：
```
http://127.0.0.1:8000/admin
```
